Video & Audio in HTML

Main rule: Multiple formats + captions

1. Video with multiple formats

            <video controls width="600" height="338" poster="preview.jpg">
    <source src="video.webm" type="video/webm">
    <source src="video.mp4" type="video/mp4">
    <p>Fallback text</p>
</video>

2. Audio player

            <audio controls preload="none">
    <source src="audio.mp3" type="audio/mpeg">
    <source src="audio.ogg" type="audio/ogg">
    <p>Fallback</p>
</audio>

3. Video with captions (WebVTT)

<track kind="subtitles" 
       src="captions-en.vtt" 
       srclang="en" 
       label="English" 
       default>

WebVTT file example:

            WEBVTT

Using WebVTT format for subtitles

4. Common Mistakes

<video src="video.mp4"></video> ← No controls! <video controls><source src="video.avi"></video> ← AVI not supported! <video autoplay></video> ← Annoying for users! <audio></audio> ← Invisible without controls!

5. Key Attributes Guide

  • controls – adds play/pause/volume (ESSENTIAL)
  • preload – "none"|"metadata"|"auto"
  • autoplay – use sparingly, requires muted on mobile
  • loop – restarts when finished
  • muted – starts without sound
  • playsinline – required for iOS inline playback
  • poster – preview image for video

6. Quick Checklist

  • Multiple formats provided?
  • Controls visible?
  • Captions/transcript available?
  • Tested on mobile?
  • Files compressed?
  • Fallback content for old browsers?
  • Copyright cleared?

Creating Hyperlinks

Main rule

Link text should describe the destination. Never use generic phrases like "click here".

List of examples

Example 1: Link Text Quality

Bad: click here to read our privacy policy.

Problem: Screen reader users hear "click here" without context.

Good: Read our privacy policy.

Better: The link text describes exactly what it leads to.

Even better: Privacy Policy

Best: Descriptive text + helpful title attribute.

Example 2: URL Path Types

Assuming current URL: https://example.com/blog/article.html

Type Example href Goes to
Absolute https://google.com Google.com
Relative (same folder) comments.html example.com/blog/comments.html
Relative (parent folder) ../index.html example.com/index.html
Root-relative /contact example.com/contact
Anchor #comments Same page, comments section

Example 3: Navigation Menu

  • Home
  • Products
  • About Us
  • Contact
  • Blog ↗

Key points:

  • <nav> for major navigation blocks
  • aria-label describes the navigation purpose
  • aria-current="page" indicates current page
  • External link has proper rel attributes
  • ↗ symbol indicates external link (visual cue)

Example 4: Special Links

Email Link

Email Support

<a href="mailto:support@company.com?subject=Help Request&body=Hello,">Email Support</a>

Telephone Link

Call: (555) 123-4567

<a href="tel:+1-555-123-4567">Call: (555) 123-4567</a>

Download Link

Download User Manual (PDF)

<a href="/files/manual.pdf" download="manual.pdf">Download User Manual (PDF)</a>

Button Link

Sign Up Free

<a href="/signup" class="button">Sign Up Free</a>
